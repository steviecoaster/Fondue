Update-ChocolateyMetadata

SYNOPSIS

Updates metadata fields in a Chocolatey .nuspec file.

SYNTAX

__AllParameterSets

Update-ChocolateyMetadata [-Metadata] <hashtable> [-NuspecFile] <string> [<CommonParameters>]

ALIASES

None

DESCRIPTION

Update-ChocolateyMetadata accepts a hashtable of metadata key/value pairs and writes each value into the matching element of the specified .nuspec file. Keys must correspond to valid nuspec element names (e.g. 'version', 'authors', 'releaseNotes'). Only the elements present in the hashtable are modified; all other elements in the file remain untouched.

The function accepts pipeline input, making it easy to chain with Convert-Xml.

EXAMPLES

EXAMPLE 1

Bump the version and add a release note

$updateMetadataSplat = @{
    NuspecFile = 'C:\packages\myapp.nuspec'
    Metadata   = @{
        version      = '2.2.0'
        releaseNotes = 'Fixed a critical bug.'
    }
}

Update-ChocolateyMetadata @updateMetadataSplat

EXAMPLE 2

Copy metadata from one nuspec into another via the pipeline

$updateMetadataSplat = @{
    NuspecFile = 'C:\packages\target.nuspec'
}

Convert-Xml -File 'C:\packages\source.nuspec' | Update-ChocolateyMetadata @updateMetadataSplat
